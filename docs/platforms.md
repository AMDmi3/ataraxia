## Supported platforms
Ataraxia Linux has been ported to 11 different CPU architectures:
```
 * x86_64       - for 64-bit x86 CPUs
 * i586         - for 32-bit x86 CPUs beginning with classic Intel Pentium
 * aarch64      - for 64-bit ARM CPUs
 * armv7hnl     - for 32-bit ARM CPUs beginning at ARMv7-a (hard-float with neon)
 * armv7hl      - for 32-bit ARM CPUs beginning at ARMv7-a (hard-float)
 * armv5tel     - for 32-bit ARM CPUs beginning at ARMv5   (soft-float)
 * ppc64le      - for 64-bit PowerPC CPUs (little-endian)
 * ppc64        - for 64-bit PowerPC CPUs (big-endian)
 * ppc          - for 32-bit PowerPC CPUs
 * riscv64      - for 64-bit RISC V CPUs
 * riscv32      - for 32-bit RISC V CPUs
```
